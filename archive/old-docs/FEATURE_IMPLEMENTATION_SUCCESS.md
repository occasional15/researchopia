### 🎯 搜索策略

#### DOI 优先搜索
```
论文有 DOI → 使用 DOI 精确搜索 → 更高准确率
论文无 DOI → 使用标题搜索 → 涵盖更多结果
```

#### 网站适配规则
- **Sci-Hub**：直接 DOI 访问（如：`sci-hub.ru/10.1000/xxx`）
- **CrossRef**：DOI 查询专用
- **PubMed**：生物医学领域优化
- **arXiv**：预印本论文专用搜索
- **中文网站**：特殊编码处理
